# Storefront Child Theme

A child theme for the Storefront WooCommerce theme.

## Features
- Preserves all parent theme functionality
- Safe from parent theme updates
- Custom styling capabilities
- WooCommerce integration
- WP Code compatible for advanced customizations

## Installation
1. Upload the `storefront-child` folder to `/wp-content/themes/`
2. Activate the theme through the WordPress admin panel

## Customization
- Add custom CSS to `style.css`
- Add custom PHP functions to `functions.php`
- Use WP Code plugin for snippet management
- Override parent theme templates by copying them to this child theme
